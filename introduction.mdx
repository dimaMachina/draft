Welcome to the Mintlify Component Demo — your single reference for every component available in Mintlify's documentation platform. Whether you're building your first docs site or looking for the right component to present complex information, this site shows every element in action with real, copy-ready code.

<Tip>
  Use the sidebar to jump directly to any component category. Each section contains live examples alongside the MDX source so you can see exactly what to write.
</Tip>

## How to Use This Demo

<Steps>
  <Step title="Browse the component categories">
    Use the sidebar or the cards below to navigate to a component section. Each page focuses on one family of components — callouts, code blocks, cards, and more.
  </Step>
  <Step title="Read the live example">
    Every section renders the component exactly as it appears in a real Mintlify docs site. What you see here is what your readers will see.
  </Step>
  <Step title="Copy the MDX source">
    Each example includes a fenced code block showing the exact markup used to produce it. Copy it, paste it into your own `.mdx` file, and adapt as needed.
  </Step>
  <Step title="Deploy and iterate">
    Drop the components into your project, run `mintlify dev` to preview locally, and push when you're happy. Refer back here any time you need a refresher.
  </Step>
</Steps>
